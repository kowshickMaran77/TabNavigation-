Introduction to the Web Workshop

Welcome to the Web Workshop! In this 4-week program, you will learn the fundamentals of web development, including HTML, CSS, and JavaScript. Each week will focus on different aspects of web development, allowing you to build your skills progressively. By the end of the workshop, you will have the knowledge to create your own web projects!

Week 1: HTML Basics

In Week 1, you will learn about HTML structure, elements, and how to create a basic webpage.

Week 2: CSS Basics

In Week 2, you will learn how to style your webpage using CSS, including layouts, colors, and fonts.

Week 3: JavaScript Basics

In Week 3, you will learn JavaScript basics and how to manipulate the DOM to make your pages interactive.

Week 4: Project and Review

In Week 4, you will apply what you've learned by building a project and reviewing key concepts.
